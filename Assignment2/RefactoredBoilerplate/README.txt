Instructions:
------------------------------------------------------------

There are 3 states:

Press F to enter foreground selection state

After pressing F:
Press the a number key from 1 to 7 to display each image:
1 - mandrill
2 - uclogo
3 - aerial
4 - thirsk
5 - pattern
6 - bubble
7 - hourglass

Press B to enter background selection state

After pressing B
Press the a number key from 1 to 3 to apply each background:
1 - asphalt
2 - marble
3 - wood

Press T to select filter
Press the a number key from 1 to 3 to apply each filter:
1 - vertical sobel
2 - horizontal sobel
3 - unsharp mask

For grey scale color change, press 8, 9, or 0:
8 - L1
9 - L2
0 - L3
