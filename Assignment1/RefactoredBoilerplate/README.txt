Instructions:
------------------------------------------------------------

Each of the scenes are mapped to a certain number,
press the a number key from 1 to 5 to see each scene:

1 - Squares and Diamonds
2 - Parametric Spiral
3 - Sierpinski Triangle
4 - Sierpinski Triangle Reloaded
5 - Barnsley Fern

For each of these scenes, 
please use the UP & DOWN arrow keys to increase/decrease 
the number of iterations displayed.
